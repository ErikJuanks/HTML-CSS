Aqui, vamos aprender sobre as Classes e os identificadores!

Classes

Classes abrangem os elementos em geral para o CSS, usada caso eu queira que um grupo de elementos se comporte com as mesmas configurações!

identificadores

identificadores são aqueles que tornam um único elemento específico, tornando-o mais fácil de configurar de maneira individual. Usado també, para evitar o uso de CSS in-line
